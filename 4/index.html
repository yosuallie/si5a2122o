Kesimpulan atau hasil pertemuan 4 yang saya ambil adalah:

  1. Langkah pertama aktifkan apache dan MySql pada Xampp
  2. Kemudian buka di browser http://localhost/si5a2122o dan di tab baru http://localhost/si5a2122o/wp-admin
  3. Lalu login pada wp-admin dengan nama admin dan password admin
  4. Page atau halaman hampir sama dengan post tetapi page bersifat tetap dan jarang diupdate seperti visi,misi,struktur organisasi,tentang perusahaan,kontak dan lokasi sedangkan post sangat sering diupdate dan sering berubah
  5. Untuk membuat page
    -klik logo w
    -lalu pilih laman tambah baru
    -(isi nama laman)setelah selesai klik terbitkan Penggunaan page sama dengan post yang membedakannya laman page tidak memiliki kategori
  6. Kemudian untuk menampilkan page
    -klik logo w
    -pilih tampilan dan klik menu
    -setelah itu pilih menu utama horizontal
  7. Untuk menghapus item menu :
    -Pilih item menu yang akan dihapus
    -kemudian klik singkirkan,Maka item menu berhasil dihapus
  8. Untuk menambah item menu: -kemudian klik ceklis pada tambah item baru dan item baru berhasil ditambhkan
    -kita juga bisa mengatur penempatan untuk menunya dengan cara di drag
  9. Untuk membuat link menjadi menu pada tampilan menu dengan cara pilih taut tersuai kemudian isi alamat url dan tekt tautannya kemudian pilih tambahkan ke menu.Maka akan menjadi item menu
  10. Untuk mengatur widget klik tampilan pilih widget(lokasi widget akan berbeda pada setiap tema yang dipilih)
  11. Untuk menginstal plugin classic widget klik plugin pilih tambah baru kemudian klik ungah plugin lalu pilih file zip yang sudah didownload kemudian klik open dan klik instal sekarang kemudian aktifkan maka widget versi lama berhasil diterapkan.(widget ditentukan oleh tema yang diaktifkan)
