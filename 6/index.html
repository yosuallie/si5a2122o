Kesimpulan atau hasil pertemuan 6 yang saya ambil adalah:

  1. Plugins merupakan suatu program mini yang dimasukan kedalam wordpress sehingga wordpress memiliki fitur-fitur tambahan contohnya seperti visitor counter.
  2. Dalam menginstal plugin dapat dilakukan melalui 3 cara yaitu dengan
  3. • Langsung online melalui wordpress atau https://wordpress.org/plugins/
    • Semi online,Dengan mengungah file zip plugin yang telah di download
    • Memindahkan folder plugins dari hasil download yang sudah di extract
  4. Cara menerapkan plugins contohnya visitor counter untuk wordpress dengan cara:
    5. • Ketikan” plugin visitor counter wordpress” di Google
    • Kemudian pilih plugins yang mendapat ulasan terbaik
    • Download plugins yang sudah dipilih,setelah berhasil di download
    • Extract file tersebut
    • Kemudian pindahkan folder yang sudah di extract ke dalam htdocs -> Folder Aplikasi->wp-contens->Plugin
    • Jika sudah akses pada halaman backend klik plugins
    • Cari Nama plugins yang dipilih tadi
    • Klik aktifkan
    • Pergi ke tampilan pilih widget
    • Kemudian drag nama plugin tadi ke dalam footer/siderbar ,maka plugin berhasil diterapkan
  5. Cara download plugins Wordfence untuk wordpres
    6. • Download file wordfence
    • Kemudian extract file hasil download Folder Aplikasi->wp-contens->Plugin
    • Masuk ke dalam akses halaman backend
    • Ke menu plugins,cari nama wordfence
    • Klik aktifkan,makan akan terdapat tambahan menu wordfence pada setting
  6. Cara Download plugins SEO Yoast
    7. • Cari pada search google dengan kata “ SEO Yoast”
    • Lalu download file SEO Yoast
    • Extrack hasil download ,kemudian masukan ke dalam htdocs -> Folder Aplikasi->wp-contens->Plugin
    • Akses halaman backend,lihat pada menu plugins
    • Maka akan muncul nama plugins SEO Yoast
    • Klik aktifkan pada SEO Yoast
    • Buka pada SEO Yoast
    • Klik configuration wizard
    • Maka akan muncul tampilan “Panduan instalasi Yoast SEO for Wordpress”
    • Pada tahap pertama (Enviroment)pilih option A
    • Pada tahap kedua (site type) pilih A blog
    • Pada tahap ketiga(organization or person)pilih organisasi, isi bagian pada kolom
    • Pada tahap ke 4(Search engine visibility) beri izin untuk perizinan
    • Pada tahap ke 5(Multiple authors) Pilih pilihan “Yes”apabila website kita memiliki banyak penulis
    • Pada tahap ke 6(Title settings)isi website name
    • Pada tahap ke 7 pilih salah satu pilihan
    • Pada tahap ke 8 pilih next
    • Pada tahap ke 9 SEO Yoast berhasil di instalasi
  7. Cara menerapkan plugins “WP Super Cache”
    8. • Download file WP Super Cache
    • Extract file hasil download
    • Pindahkan folder hasil extract ke dalam htdocs -> Folder Aplikasi->wp-contens->Plugin
    • Buka halaman backend,Cari nama plugin WP Super cache klik aktifkan
    • Pergi ke menu setting terdapat option untuk WP Super cache
    • Pada Caching pilih Caching on kemudian update status
  8. Membuat Contact Form
    9. • Pilih Everest forms pada halaman backend
    • Klik add new
    • Pilih Start from scatch
    • Masukan Nama form,klik continue
    • Tambahkan single line text,kemudian drag
    • Untuk menambah umur pilih date/time kemudian drag
    • Untuk menambah kolom nomor pilih number kemudian drag
    • Kemudian klik save
    • Untuk mengunakan form tersebut menggunakan shortcode yang sudah di dapat,kemudian copy
    • Tambah halaman baru,masukan nama judul
    • Kemudian untuk isi halaman paste kan shortcode yang dicopy sebelumnya
    • Untuk menampilkan halaman yang dibuat ,jadikan halaman tersebut sebagai menu
    • Dari tampilan menu,pilih nama halaman kemudian ceklis,tambahkan ke menu
    • Kemudian klik simpan
